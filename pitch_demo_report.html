Q21: What is superposition in neural networks?

Baseline (no knowledge)
Superposition in neural networks refers to the phenomenon where a single neuron or layer represents multiple different features simultaneously. This happens because neural networks tend to learn efficient representations that pack more information into fewer dimensions. It makes interpretability harder because features can't be cleanly isolated to individual neurons.
Acc: 4/5 Spec: 3/5 Edu: 3/5 Avg: 3.3
Pharos (knowledge injected)
Superposition occurs when models represent more features than dimensions: a network with n activation dimensions can represent significantly more than n features by encoding each feature as a direction in activation space rather than assigning one feature per neuron. Features are represented as near-orthogonal vectors, and because most features are sparse (rarely active), superposition is recoverable — the interference between simultaneously active features stays low on average. This is why neuron-level interpretability fails: a single neuron participates in representing many features, and a single feature is distributed across many neurons.
